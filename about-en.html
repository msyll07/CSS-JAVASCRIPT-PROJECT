~ Project Name: EasyStyle

Purpose: EasyStyle aims to provide developers with a fast and flexible CSS framework. With specific components and understandable classes, it offers a quick and modern web development experience.

~ Class Naming Convention:

To ensure that the framework is easy to use and understand, we will adopt a simple and consistent class naming convention. All classes will be prefixed with es- (short for EasyStyle) to avoid conflicts with other frameworks or custom styles.
Example class names:

  • es-btn
  • es-form
  • es-navbar
  • es-card .....

~ Project Proposal Details:

EasyStyle will feature a comprehensive documentation website similar to Bootstrap's documentation site. This site will showcase all the components, grid system, and utility classes available in the framework. Each component will have a dedicated page with examples, usage instructions, and customization options.
